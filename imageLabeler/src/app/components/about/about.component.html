Purpose

This visualization app showcases my personal career interests and skills. I was working for a medical company that focused on fMRI post-processing that was moving from desktop to web-based visualization. We were very pleasantly surprised by the speed at which we could build our web application with vtk.js as well as the rendering speeds it offers. This prototype is a reflection of about 3 weeks of work.

This visualization app is a prototype for labelling regions of a MRI scan. It is a work-in-progress looking at optimizing the interactive interfaces for marking regions of a 3D volume.

A more mature desktop application of the same nature is the VTK Slicer desktop application. I am pursuing looking more deeply at the human interface options for manual or semi-automatic segmentation.

Technologies

This visualization app leans heavily on vtk.js which provides the volume rendering and 3D reslicing visualizations.

The UI uses Angular for the overall user interface along with Angular Materials as well as RxJS for the overall integration of the User Interface.

The server is NodeJS using the DICOM parser npm package dicom-parser. The server uses Koa and it's router, body-parser, and static add-ons.

How to Use

  1. Select an exam (either Exam 008 or Exam 25. by clicking on the "Exam" text.
  2. Select a series under the exam just selected by clicking on the numerical items.
  3. The series will take a while to load. I am using a budget server, and the data is about 0.0 Gigs for a typical dataset. Once loaded, the 2 viewports appear with images.
  4. Adjust the view around some area of interest to trace in the left viewport. Initially the mouse mode will be set to "WL" where you can adjust the brightness and contrast. Mouse drag left/right adjusts contrast. Mouse drag top/bottom adjusts brightness with mouse-down increasing brightness. Change mouse modes to either of PAN for moving the image, ZOOM for adjusting magnification. To increase magnification, mouse down below the center of the image and drag down. Similarly, drag up for reducing magnification. PAGE is to scroll through the stack of images of the volume.
  5. Start drawing by selecting DRAW and drag the mouse around the region of interest, filling in the region. Then move to the next slice by using the scroll wheel. Continue through the stack of slices. You can switch to the ERASE mode to erase some unwanted drawn region.
  6. Adjust the global reference volume rendering to gain a feel for the relative region from a more 3D perspective. In the global region, mouse drag will rotate the view. The scroll wheel will perform the zoom adjustment. The pan requires holding the shift key while dragging the mouse.

Source of DICOM Data

  1. Go to central.xnat.org
  2. Pick [MR] tab
  3. Under 'Project:' field, click the field and a list of projects is presented.
  4. Select IGT_FMRI_NEURO
  5. Click the [Submit] button.
  6. Pick a session such as label [025 IGT_FMRI_NEURO) for 025
  7. Select all the series You will get the series as a zop file that contains DICOM part 10 files. Unpack the files and put them into a directory structure
